## 🔧 Installation

### Prerequisites

- Node.js 18.0 or later
- npm 8.0 or later
- Git

### Quick Start

```bash
# Clone the repository
git clone https://github.com/tiation/tiation-github-pages-theme.git
cd tiation-github-pages-theme

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
